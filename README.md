# supabase-votes-app

تطبيق بسيط لإدارة التصويت باستخدام Supabase وVercel.

## المميزات
- إضافة تصويت جديد عبر POST request
- عرض جميع الأصوات عبر GET request
- متصل بقاعدة بيانات Supabase

## التنصيب
1. انسخ المشروع
2. قم برفعه على Vercel
3. اضبط متغيرات البيئة في Vercel

## الروابط
- API: `/api/votes`

## المتغيرات البيئية المطلوبة
- RECAPTCHA_SECRET_KEY
